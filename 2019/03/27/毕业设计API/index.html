毕业设计API

发表于 2019-03-27

IP+端口：127.0.0.1:8080
context-path: /gameshop

test: ../hello

用户 /user

  • 登录：
    url：/signin
    parameter: User
    response：User

  • 注册：
    url: /signup
    parameter: User
    response：success

游戏 /game

  • 获取所有游戏
    url: /listgame
    parameter；none
    response：games

  • 游戏详细信息
    url: /detail
    parameter: Interger GID
    response：game

  • 获取某类型的所有游戏
    url: /type
    parameter: Interger type
    response：games

游戏类型参考表：

数值 游戏类型
0 未定义类型
1 FPS游戏

库存 /repository

  • 查询用户的所有订单
    url: /repository
    parameter: Interger UID
    response：repositories

  • 购买（添加订单，修改库存）
    url: /buy
    parameter: Game
    response：success

错误返回

response：
success: false
